smooth

slightly cinematic

Build notes for the developer

Tell the developer to follow these rules:

Build this as a single-page React + TypeScript site.

Use Sass modules or structured SCSS, not Tailwind.

Use Motion for React for all major motion and reveal behavior.

Use Lenis for smooth scrolling, but keep accessibility and reduced-motion fallbacks.

Use Radix Dialog for project modals.

Prioritize premium restraint over spectacle.

Reduce motion on mobile and for prefers-reduced-motion.

Optimize all imagery and video previews carefully.

Keep interaction polish high:

hover states

focus states

transitions

keyboard accessibility

Ensure the site still feels strong with motion disabled.
